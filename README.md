# Анализ событий информационной безопасности

## Описание проекта
Анализ и визуализация распределения событий информационной безопасности по их типам.

## Используемые библиотеки
- pandas
- matplotlib
- seaborn
- numpy

## Структура проекта
- `analyze_security_events.py` - основной скрипт анализа
- `requirements.txt` - зависимости
- `events.json` - пример данных
- `security_events_analysis.png` - результаты визуализации

## Запуск анализа
```bash
# Установите зависимости
pip install -r requirements.txt

# Запустите анализ
python analyze_security_events.py
